Flavourful Nkatie Cracks Website README
Overview
A modern multi-page snack brand website built for Flavourful Nkatie Cracks, featuring Vanilla, Chocolate, and
Strawberry flavoured peanut snacks in multiple package sizes.
Features
• Responsive mobile-friendly design
• Multi-page layout
• Product showcase section
• Gallery page with promotional visuals
• WhatsApp ordering integration
• Modern UI styling
• Fast-loading static website
• Hosted using GitHub Pages
